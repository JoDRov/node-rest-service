## INSTRUCTIONS

1. Type: "npm i" to install every dependency to run our program
2. Type: "npm start" to start the server

# From here you can open your browser and go to http://localhost:3001/ to see the main page or http://localhost:3001/api/notes/ to check all the notes.

# If you install the extension "REST client" you can use the scripts on the folder request to create, change status or delete notes by clicking on "Send request" above each request.

# Also you can use Insomnia to create, change status, delete or check the notes
